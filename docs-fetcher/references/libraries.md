## How to Add More Libraries

Add new entries following this format:

```markdown
### Library Name

| Topic | URL |
|-------|-----|
| Topic Name | https://example.com/docs/topic |
```

**Tips:**

- Prefer official documentation over third-party tutorials
- Include quickstart/getting-started pages
- Add topic-specific pages for common use cases
- Test URLs before adding (some may require authentication or block bots)
